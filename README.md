Todolist Web App

A lightweight and interactive web-based task management application built using Vanilla JavaScript. This application helps users organize their productivity with real-time task prioritization and deadline tracking.

Key Features
- Task Management (CRUD): Add new tasks and delete entire lists with a single click.
- Priority System: Classify tasks by importance (Low, Medium, High) with visual color indicators.
- Deadline Tracking: Enter a due date and automatically separate overdue tasks into the "Overdue" category.
- Status Grouping: Tasks are dynamically organized into three main categories:
- Todo: Active task list.
- Completed: Tasks that have been completed (cross-out toggle feature).
- Overdue: Tasks that are past their due date.
- Dark Mode Support (Dark Mode): Adaptive interface that adapts to the user's operating system theme preferences.

Technologies Used
- HTML5: Semantic structure for better accessibility.
- CSS3: Responsive design using Flexbox and implementing Dark Mode via the prefers-color-scheme media query.
- JavaScript (ES6+): Pure DOM manipulation without external libraries to keep performance light.

Project Structure
- index.html - The main framework of the application.
- index.css - Application styling, including responsive layout and color themes.
- index.js - The main logic of the application, including dynamic element creation and time-of-day validation.

How to measure
- Clone this repository to your local computer.
- Open the index.html file using your preferred browser.
- The application is ready to use!
